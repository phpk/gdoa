

## 说明



## 安装

```bash
$ npm install
```

## 执行

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
## 生成文档

```bash
npm run doc
```

## 快速创建

```bash
nest g mo 模块名称
nest g res CRUD
nest g cl 类名
nest g s service服务
```

## 测试

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
